# OSINT & Social Engineering: Information Gathering 🕵️‍♂️🌐

## Overview
This repository contains the findings and report for the **BAXZ3613 - Hacking Techniques and Prevention** course (Assignment 1: Hacking the Human). The objective of this project was to utilize Open-Source Intelligence (OSINT) and information gathering techniques to trace and compile digital footprints and personal data of a specific target[cite: 9].

## Project Scope & Objectives
The assignment was divided into three progressively challenging sections[cite: 9]:
1. **Section A (The Normies):** Gathering basic public information (e.g., Full Name, Identity Card Number, Permanent Address, Personal Email, Social Media)[cite: 9].
2. **Section B (Advanced):** Discovering secondary layer information (e.g., Work/Staff Emails, Current Address, Secondary Social Media)[cite: 9].
3. **Section C (The Chosen One):** A deep-dive challenge to uncover educational history (Matric Numbers, CGPA, Graduation Years) and hidden online personas/accounts (e.g., Discord, YouTube, Instagram)[cite: 9].

## Tools & Techniques Utilized
* **OSINT Frameworks:** Leveraged various web-based OSINT tools (e.g., WhatsMyName) to cross-reference usernames and discover hidden social media accounts[cite: 9].
* **Public Directories & Portals:** Utilized university portals and public directories to extract academic and professional information[cite: 9].
* **Social Media Analysis:** Mapped digital footprints across LinkedIn, Facebook, YouTube, Instagram, and Discord[cite: 9].
* **Communication Platforms:** Extracted organizational emails via platforms like Outlook[cite: 9].

## Key Findings Summary
* Successfully identified core identity details including Full Name, IC Number, and precise residential addresses[cite: 9].
* Mapped professional and academic progression, including specific Matriculation Numbers (Degree & PhD) and academic performance (CGPA)[cite: 9].
* Uncovered a wide array of digital communication channels, including primary/secondary emails, mobile numbers, and active online aliases across multiple platforms (LinkedIn, Facebook, Discord, YouTube, Instagram)[cite: 9].
